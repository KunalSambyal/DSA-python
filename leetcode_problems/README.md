# LeetCode Problems Solutions

This directory contains solved LeetCode problems organized by topic and difficulty, structured to follow the NeetCode and Striver A2Z roadmaps.

---

## Problem Categories

Currently documented problem sets:

- **01 Arrays & Hashing:** Solutions for frequency maps, prefixes, and basic array validations.
    - [contains_duplicate.py](01_arrays_and_hashing/0217_contains_duplicate/contains_duplicate.py) - Check if array contains duplicates.
    - [valid_anagram.py](01_arrays_and_hashing/0242_valid_anagram/valid_anagram.py) - Check if two strings are anagrams of each other.
    - [two_sum.py](01_arrays_and_hashing/0001_two_sum/two_sum.py) - Find indices of two numbers that add up to a target.
- **02 Stacks:** Solutions for sequential validation, reverse order processing, and stack usage.
    - [valid_parentheses.py](02_stacks/0020_valid_parentheses/valid_parentheses.py) - Validate parenthetical bracket balance in a string.
