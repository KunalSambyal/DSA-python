# DSA Basics

This folder contains references, implementations, and study notes for fundamental DSA building blocks.

---

## Core Topics Covered

- **01 Complexity Analysis:** [01_complexity_analysis](01_complexity_analysis/README.md) - Understanding Big-O runtime notation, space constraints, and scaling analysis.
- **02 Basic Mathematics:** [02_basic_math](02_basic_math/README.md) - Digit extraction, palindrome checks, GCD/LCM, divisors, and primality testing.
- **03 Basic Recursion:** [03_basic_recursion](03_basic_recursion/README.md) - Call stacks, base cases, linear recursion, factorial/Fibonacci, array/string recursive traversals.
- **04 Basic Hashing:** [04_basic_hashing](04_basic_hashing/README.md) - Frequency mapping, count queries, and finding min/max frequency elements.
- **05 Loop Patterns:** [05_patterns](05_patterns/README.md) - Nested loop practices with symmetrical star and number layouts.
