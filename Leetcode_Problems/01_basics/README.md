# 01 Basics

This folder contains my solutions and notes for introductory concepts and basic problem solving.

## Key Concepts to Cover

- **Time & Space Complexity (Big-O Notation):** Understanding how runtime and memory scale with input size.
- **Basic Mathematics:** Operations on digits, Prime numbers, Greatest Common Divisor (GCD) / Least Common Multiple (LCM), finding divisors, and basic number theory.
- **Basic Recursion:** Understanding the call stack, base cases, and writing simple recursive functions.
